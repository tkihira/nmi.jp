よく使うショートカット windows編

2012-06-26
Takuo Kihira
Tweet

Windowsで普段自分がよく使うショートカットを列挙してみます。
あくまでもよく使うものに限ります。思い出したら追記していきます。

ALT+TAB：タスク切り替え（ALT+SHIFT+TABで逆順）
CTRL+SHIFT+ESC：タスクマネージャ起動
ALT+F4：Windowを閉じる

Windows + [D]：デスクトップ表示
Windows + [E]：エクスプローラ起動
Windows + [R]：ファイル名を指定して実行(cmd, calcが多い。notepad, pbrush, regedit等も)

ALT+[SPACE]、X：ウィンドウを最大化
ALT+[SPACE]、R：ウィンドウを元の大きさに戻す
ALT+[SPACE]、M：ウィンドウを移動（キーボードで）

Explorer/IE/ブラウザ
[F2]：フォーカスがあるところの名前変更
SHIFT+DEL：ゴミ箱に入れずに消去
[F5]：アタック(CTRL + [R]も同様)
[F6]：アドレスバーに移動（正確にはフレーム移動）
ALT+HOME：ホームページに移動
ALT+[←]：前のページに戻る（[BS]でも同様）
ALT+[→]：次のページに行く
TAB: フォーカス移動

CTRL+[リンククリック]：別のタブでリンクを開く
SHIFT+[リンククリック]：別のウィンドウでリンクを開く
CTRL+[T]：新しいタブ
CTRL+SHIFT+[T]：直前に閉じたタブの復元（Chrome & Firefox）

大抵のアプリで同じ挙動
CTRL + [C]：コピー
CTRL + [X]：カット
CTRL + [V]：ペースト
CTRL + [A]：全選択

CTRL + [N]：新規作成
CTRL + [S]：保存
CTRL + [F]：検索

CTRL + [Z]：UNDO
CTRL + [Y]：REDO(UNDOの取り消し)

CTRL + [W]：タブ もしくは子Windowを閉じる
