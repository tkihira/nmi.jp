iOS5 browser new features: position-fixed, overflow-scroll, etc...

2011-10-14
Takuo Kihira
HTML5  SVG  Canvas  JavaScript  Smart Phone
Tweet

You can check iOS5 new features: http://nmi.jp/examine/
iPhone 4Sをいろいろと触り倒しているなかで、噂の新しい機能を色々と試してみたのでブログに残しておきます。

iOS5の新しい機能は結構あるのですが、実機で私が試したのは次の一覧です。

  • CSSのposition:fixedへの対応
  • CSSのoverflow:scrollへの対応
  • HTML5のinput type="日付系"への対応
  • HTML5のinput type="range"への対応
  • HTML5のWeb Workersへの対応
  • inline SVGへの対応
  • HTML5のCanvas描画の高速化

特に上の2つは、今まで苦労して対応していたレイアウトが一気に解決する可能性を持った嬉しい対応になると思います。Web Workersへの対応は個人的にはとても嬉しいですね。inline SVGは書くのが楽になってこれも嬉しい。最後のCanvasの高速化は、最高に嬉しい対応です。

これらを組み合わせると、今まで出来なかった新しいWebアプリの可能性が広がるのではないかと期待しています。さっそく私もいろいろと作ってみたいと思います。

上のリストから、Canvas高速化以外の機能を試すページを作りました。ぜひ実機から試してみてください。
http://nmi.jp/examine/
